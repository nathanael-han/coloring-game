How to play:

Click on a color then mouse over the hexagons — the hexagons will change to the selected color!

Assignment 14:

One change I made to the original game was to use 6 instead of 5 sides. Using hexagons allowed me to tile the canvas without gaps — I implemented this by drawing the polygons within for loops. Instead of using the "click" event listener, I used "mouseover" to change the color when a hexagon is hovered over. I also added an animation when the colors are selected; when a rectangle is clicked, it increases in scale, then decreases back to the original size. Additionally, I resized and centered the canvas, and I used coolors.com to find a color scheme.
